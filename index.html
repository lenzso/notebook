[-] 数据类型

  • NoneNone 空值
  • int 整数，支持16进制，0xff00
  • float1.23e-5 1.23e9，1.23e-5
  • boolTrue / False True / False
    区分大小写
    在 Python2 中是没有布尔型的，它用数字 0 表示 False，用 1 表示 True。到 Python3 中，把 True 和 False 定义成关键字了，但它们的值还是 1 和 0，它们可以和数字相加。
  • complex3+4j
  • string'str' 字符串用单引号(')或双引号(")括起来，同时使用反斜杠(\)转义特殊字符。
    Python 字符串不能被改变。向一个索引位置赋值，比如word[0] = 'm'会导致错误。
    索引值以 0 为开始值，-1 为从末尾的开始位置。
    字符串截取：变量[头下标:尾下标]，不包含最后位置
    加号 (+) 是字符串的连接符
    星号 (*) 表示复制当前字符串 
    
                        str = 'ABCDEFG'
                        print (str)          # 输出字符串
                        print (str[0:-1])    # 输出第一个个到倒数第二个的所有字符
                        print (str[0])       # 输出字符串第一个字符
                        print (str[2:5])     # 输出从第三个开始到第五个的字符
                        print (str[2:])      # 输出从第三个开始的后的所有字符
                        print (str * 2)      # 输出字符串两次
                        print (str + "TEST") # 连接字符串
  • tuple(1,'a') 元组（tuple）与列表类似，不同之处：
    1.元组的元素不能修改
    2.元组写在小括号()里
    3.空元组 tup1 = ()
    4.一个元素，需要在元素后添加逗号 tup2 = (20,) 否则非元组，而是单元素
    5.tuple的元素不可改变，但它可以包含可变的对象，比如list列表。
  • list[1,'a'] 列表是写在方括号[]之间、用逗号分隔开的元素列表。
    List中的元素是可以改变的。
    索引值以 0 为开始值，-1 为从末尾的开始位置。
    列表截取：变量[头下标:尾下标]，不包含最后位置
    加号 (+) 是字符串的连接符
    星号 (*) 表示复制当前字符串 
    
                        list = [ 'abcd', 786 , 2.23, 'runoob', 70.2 ]
                        tinylist = [123, 'runoob']

                        print (list)            # 输出完整列表
                        print (list[0])         # 输出列表第一个元素
                        print (list[1:3])       # 从第二个开始输出到第三个元素
                        print (list[2:])        # 输出从第三个元素开始的所有元素
                        print (tinylist * 2)    # 输出两次列表
                        print (list + tinylist) # 连接列表
  • set{'a','b'}/set('aabc') 集合（set）是一个无序不重复元素的序列。基本功能是进行成员关系测试和删除重复元素。
    可以使用大括号({})或者 set()函数创建集合。
    创建一个空集合必须用 set() 而不是 { }，因为 { } 是用来创建一个空字典。
    使用set()时最多传入1个参数。
    运算符 结果
    - 前者存在后者不存在
    ^ 不同时存在的元素
    | 并集
    & 交集
  • dictionary{'k':'v'}/dict() 列表是有序的对象结合，字典是无序的对象集合。两者之间的区别在于：字典当中的元素是通过键来存取的，而不是通过偏移存取。字典是一种映射类型，字典用"{ }"标识，它是一个无序的键(key) : 值(value)对集合。键(key)必须使用不可变类型。在同一个字典中，键(key)必须是唯一的。
    内置函数：clear()清空、keys()键组、values()值组。 
    
                        tinydict = {'name': 'test','code':1, 'site': 'www.test.com'}
                        print (tinydict.keys())   # 输出所有键
                        #tinydict_keys(['name', 'site', 'code'])
                        print (tinydict.values()) # 输出所有值
                        #tinydict_values(['test', 'www.test.com', 1])
    构造函数 dict() 可以直接从键值对序列中构建字典如下： 
    
                        >>> dict([('Runoob', 1), ('Google', 2), ('Taobao', 3)])
                        {'Taobao': 3, 'Runoob': 1, 'Google': 2}

                        >>> dict(Runoob=1, Google=2, Taobao=3)
                        {'Taobao': 3, 'Runoob': 1, 'Google': 2}

                        >>> {x: x**2 for x in (2, 4, 6)}
                        {2: 4, 4: 16, 6: 36}

[-] 数学函数

  • max(x,y,z) 返回给定参数的最大值，参数可以为序列。
  • min(x,y,z) 返回给定参数的最小值，参数可以为序列。
  • abs(x) 绝对值，返回数字的绝对值，如abs(-10) 返回 10
  • math.fabs(x) 绝对值，返回数字的绝对值，如math.fabs(-10) 返回10.0
  • round(x,[n]) 返回浮点数x的四舍五入值，如给出n值，则代表舍入到小数点后的位数。
  • math.ceil(x) 向上取整，返回数字的上入整数，如math.ceil(4.1) 返回 5
  • math.floor(x) 向下取整，返回数字的下舍整数，如math.floor(4.9)返回 4
  • pow(x,y,[z]) 返回 xy（x的y次方） 的值。
    如果z在存在，则再对结果进行取模，其结果等效于pow(x,y) %z
  • math.sqrt(x) 返回数字x的平方根，数字可以为负数，返回类型为实数，如math.sqrt(4)返回 2+0j
  • math.exp(x) 返回e的x次幂(ex),如math.exp(1) 返回2.718281828459045
  • math.log(x) 如math.log(math.e)返回1.0。math.log(100,10)返回2.0
  • math.log10(x) 返回以10为基数的x的对数，如math.log10(100)返回 2.0
  • math.modf(x) 返回x的小数部分与整数部分，两部分的数值符号与x相同，整数部分以浮点型表示。

[-] 随机数函数

  • random.choice(seq) 从序列的元素中随机挑选一个元素
    seq -- 可以是一个列表，元组或字符串。
    比如random.choice(range(10))，从0到9中随机挑选一个整数。
  • random.randrange([start,] stop [,step]) 从指定范围内，按指定基数递增的集合中获取一个随机数
    start -- 指定范围内的开始值，包含在范围内，缺省为0。
    stop -- 指定范围内的结束值，不包含在范围内。
    step -- 指定递增基数，缺省为1。
  • random.random() 随机生成下一个实数，它在[0,1)范围内。
  • random.seed(x) 改变随机数生成器的种子seed。
  • random.shuffle(lst) 将序列的所有元素随机排序
    lst -- 可以是一个序列或者元组。
  • random.uniform(x,y) 随机生成下一个实数，它在[x,y]范围内。
